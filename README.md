# VisionHands

**VisionHands** — это система жестового управления компьютером на базе искусственного интеллекта. Проект объединяет компьютерное зрение (MediaPipe), современный графический интерфейс (PySide6) и автоматизацию ОС для создания бесшовного пользовательского опыта.

---

## Основные возможности

- **Surgical Mouse Control**: Уникальная двуручная схема управления. Правая рука отвечает за плавное и точное перемещение курсора, а левая — за клики и системные команды.
- **Context-Aware Profiles**: Система автоматически распознает активное окно (например, YouTube) и переключает набор жестов.
- **Gesture Authentication**: Многоэтапная система входа для защиты от случайных срабатываний.
- **Persistent Settings**: Интеграция с базой данных SQLite для сохранения параметров чувствительности и сглаживания.
- **Modern UI**: Минималистичный футуристичный интерфейс с системным треем и прозрачными элементами.

---

## Управление

### Авторизация 
1. `POINTER` (Указательный)
2. `OK_SIGN` (Жест ОК)
3. `PEACE` (Два пальца)
4. `FIST` (Кулак)
*Удерживайте каждый жест 1.5 сек для подтверждения.*

### Операционная система
- **Правая рука - Указательный**: Передвижение курсора.
- **Левая рука - Указательный**: Левый клик (ЛКМ).
- **Левая рука - Два пальца (Peace)**: Правый клик (ПКМ).
- **Два кулака вместе (Double Fist)**: Переключение окон (Alt + Tab).

### Специальный профиль: YouTube
- **Правая рука - Кулак**: Пауза / Воспроизведение (Space).
- **Левая рука - Кулак**: Следующее видео (N).

## Технологический стек

- **Core**: Python 3.10+
- **Computer Vision**: MediaPipe Hand Landmarker
- **GUI**: PySide6 (Qt for Python)
- **Database**: SQLite3
- **Automation**: PyAutoGUI, PyGetWindow

## Быстрый старт

1. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```
2. **Скачайте модель**:
   Поместите `hand_landmarker.task` в папку `models/`.
3. **Запустите приложение**:
   ```bash
   python main.py
   ```
