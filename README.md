# VisionHands — Gesture Controller

Система управления компьютером с помощью жестов рук, использующая компьютерное зрение. Проект позволяет заменять мышь движениями рук перед камерой и включает систему авторизации по "жестовому паролю".

## Основные возможности

- **Авторизация по жестам**: Доступ к управлению только после выполнения секретной последовательности жестов.
- **Управление курсором**: Точное перемещение мыши указательным пальцем правой руки.
- **Клики**:
  - Левый клик — указательный палец левой руки.
  - Правый клик — сжатие правой руки в кулак.
- **Скроллинг**: Вертикальная прокрутка при помощи двух пальцев (указательный + средний) правой руки.
- **Интерактивный HUD**: Отрисовка скелета руки, отображение текущего FPS и активных жестов в реальном времени.
- **Гибкая настройка**: Параметры сглаживания, скорости скролла и задержек кликов сохраняются в локальной базе данных.

## Технологии

- **Python 3.10+**
- **MediaPipe**: Для высокоточного отслеживания ключевых точек руки.
- **OpenCV**: Для захвата видеопотока и отрисовки интерфейса.
- **PyAutoGUI**: Для эмуляции действий мыши в системе.
- **SQLite**: Для хранения пользовательских настроек.

## Использование

Запустите проект командой:
```bash
python main.py
```

### Авторизация (Gesture Lock)
При запуске программа потребует ввести пароль. По умолчанию это последовательность:
1. **ONE** (Указательный палец)
2. **FIST** (Кулак)
3. **TWO** (Указательный + Средний)
4. **ROCK** (Указательный + Мизинец)

*Удерживайте каждый жест около 1.2 секунд до подтверждения.*

### Горячие клавиши
- `G` — Включить/Выключить управление жестами.
- `Q` или `ESC` — Выход из приложения.

### Жесты управления
| Рука | Жест | Действие |
| :--- | :--- | :--- |
| **Правая** | Указательный палец | Перемещение курсора |
| **Правая** | Указательный + Средний | Скроллинг (вверх/вниз) |
| **Правая** | Кулак | Правый клик |
| **Левая** | Указательный палец | Левый клик |

## Настройка
Пароль можно изменить в файле `gesture_auth.py`, отредактировав список `PASSWORD_SEQUENCE`.  
Настройки чувствительности и сглаживания можно изменить в `settings_manager.py` или напрямую в базе `user_settings.db`.
